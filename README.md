```Clone the Repo:
git clone https://github.com/mrlaksh20/Jscan
```
```just install:
sudo bash sertup.sh
```
```Run The Tool:
jscan https://targ.com/  n | s
```
```Note:
'n' is for normal home page of website that it will crawl, like this https://target.com/
's' is for specific path like this : https://target.com/profile/
```
